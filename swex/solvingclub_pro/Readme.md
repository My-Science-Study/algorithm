# NOTE  

## Suffix Array  

`Suffix Array`란, 문자열 `S`가 있을 때 그 접미사들을 정렬해놓은 배열이다.   

|**suffix**|  **i**  |
|:---      | :---: |
|banana|1|
|anana|2|
|nana|3|
|ana|4|
|na|5|
|a|6|

이를 `Suffix string` 순으로 정렬하면 아래와 같다.  

|**suffix**|  **i**  |
|:---      | :---: |
|a|6|
|ana|4|
|anana|2|
|banana|1|
|na|5|
|nana|3|
